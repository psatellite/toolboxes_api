Introduction

This GUI, PointingBudget, lets you interactively create a pointing budget.

File

  • New: Create a new file by typing in the file
  • Open: Open and existing file.
  • Save: Save a file.
  • _Print a file:_ Print a file.

Error Entry

Enter the roll, pitch and yaw components, the name of the error. Select
the category and click "Add". It will be added to the list on the left.
To remove an error, click the error and push "Remove"

Methodology

Select the desired methodology.

Azimuth and Elevation

Type in the beam azimuth and elevation. Hit "Update" to update the results
with the new values.
It will display the results in the list on the right.

Sort

Sort the errors.

Quit

Quit the GUI.
