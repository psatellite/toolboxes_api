Overview

PSS toolboxes provides three broad classes of tools:

  • Design
  • Analysis
  • Simulation

which can be applied to a variety of spacecraft design problems. Please see the manuals included with your toolboxes for details on the types of functions available.

Getting Started

There are two key functions you should be start with in learning your way around the toolboxes.

The first is FileHelp. This GUI enables you to browse or search the MATLAB headers of all the toolbox functions. You can also access the headers from the command line using regular MATLAB help commands, but this GUI makes browsing searching fast and efficient, especially when you are just getting started and don't know all the folders in your toolboxes yet.

The second is DemoPSS. This gathers all the demos in your toolboxes so that you can view and run them easily. There is no need to search through the directories in your toolbox for relevant demos, DemoPSS does it for you!
