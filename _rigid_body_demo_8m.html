RigidBodyDemo.m File Reference
AC Module » AC Module Demos

Simulate the linear and angular states of a rigid body using "AC.m" Published Demo More...

Detailed Description

Four different geometries are available to choose from. A rotor may be included or omitted.

The motion is simulated using two different cases. case 0 – The origin of the body frame coincides with the CG case 1 – The origin of the body frame is translated to a non-CG point

The inertias for case 0 and case 1 are different, according to the parallel axis theorem.

The initial angular velocity is different for each case IF the inertia matrix has non-zero off-diagonal terms.

No aerodynamic forces or torques are considered.

VERIFICATION

  • The trajectories of both the CG and a non-CG point on the body are computed for each case, and the errors are plotted.
  • The total angular momentum (body + rotors) is plotted. It should be constant.
