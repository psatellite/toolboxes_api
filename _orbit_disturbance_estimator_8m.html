OrbitDisturbanceEstimator.m File Reference

Demonstrates orbit disturbance estimation using linearized equations. More...

Detailed Description

Published Demo

This uses the linearized orbit equations in-plane. A radial disturbance is modeled and estimated. The equations are normalized so that the derivative is respect to wo*t instead of t. This improves the numerical properties of the state transition matrix which otherwise would cause DQCE to fail. DQCE has an inverse in it which is not particularly robust numerically. Generally, this kind of scaling is always a good idea.
