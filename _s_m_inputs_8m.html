◆ DeleteUnavailableHSAs()

function DeleteUnavailableHSAs ( in hSAs,
in avail )

Checks the userdata of the methodButton; if userdata doesn't exist or is invalid then the first output is determined by the HSAs with available data, and lastHSAs is -1. Otherwise, both hSAs and lastHSAs are the same as the userdata

Inputs

methodButton (1) Handle to the checkbox for this method

Outputs

hSAs (1) 1 if HSA 1, 2 if HSA 2, 3 if both, 0 if none avail (2) For each HSA, 1 means data is available, 0 means no data lastHSAs (1) Userdata of methodButton if it exists and is valid (has value in 0:3); -1 else
