SLPLComparisonDemo.m File Reference

Compare a constant-thrust mission to optimal linear acceleration Published Demo More...

Detailed Description

First find the constant-thrust solution for a given distance, time, and engine configuration. Then compute the equivalent optimal mission in two cases:

 1. the fuel is 100% consumed and the trip is shorter; 
 2. the trip is the same duration and the saved fuel mass can be extra
 payload.
