The Spacecraft Control Toolbox has been Princeton Satellite System's flagship product since 1995. The toolbox is organized into modules by functionality, for example Link and Orbit. The documentation, function API and published demos for each module are linked below.

SCT Modules

Core Utilities

These are general system functions and utilities.

Module Functions Demos Description
Dynamics Module Dynamics Functions Demos Example dynamics including process, robotics, plasma, and automotive.
Electrical Module Electrical Functions Demos Electromagnetism and electronics.
Common Module Common Functions Demos Utilities for control, estimation, coordinates, graphics, and time.
Imaging Module Imaging Functions Demos Optics and image processing.
Math Module Math Functions Demos Numerical utilities and math analysis.
Plotting Module Plotting Functions Demos Plotting tools for complex simulations including animation.

Spacecraft Essentials

The modules in the Spacecraft Control Toolbox are grouped by subsystem. All attitude dynamics and control functions are in SC.

Module Functions Demos Description
AerospaceUtils AerospaceUtils Functions (none) The AerospaceUtils module is part of both Aircraft and Spacecraft Control Toolboxes.
CubeSat CubeSat Functions Demos Simplified small satellite integrated model.
Link Link Functions Demos Optical and RF link analysis and budgets.
Missions Missions Functions Demos Detailed mission case studies in Earth orbit.
Orbit Orbit Functions Demos Orbit mechanics and maneuver planning in Earth orbit.
Propulsion Propulsion Functions Demos Chemical, electric and nuclear thruster modeling.
SC SC Functions Demos Spacecraft attitude dynamics and models.
SpacecraftEstimation SpacecraftEstimation Functions Demos Attitude and orbit estimation for spacecraft.
Thermal Thermal Functions Demos Heat transfer, network and isothermal models.

Interplanetary Module Pack

The interplanetary module pack extends the analysis to anywhere in the solar system. Lunar missions are grouped together, as are straight-line trajectories.

Module Functions Demos Description
Interplanetary Interplanetary Functions Demos Supporting missions beyond cislunar space.
LunarMissions LunarMissions Functions Demos Missions at or near the Moon.
StraightLine Functions Demos Straight line trajectories for advanced propulsion.

Add-On Modules

Add-on modules are available to extend SCT for specialized missions.

Module Functions Demos Description
FormationFlying FormationFlying Functions Demos Framework for reconfigurable, decentralized satellite formations.
LaunchVehicle LaunchVehicle Functions Demos Modeling and simulation of multistage launch vehicles.
OpticalNavigation OpticalNavigation Functions Demos Optical navigation using the stars and planets.
SAAD SAAD Functions Demos Spin-axis attitude determination.
Sail Sail Functions Demos Solar sail guidance and control.
