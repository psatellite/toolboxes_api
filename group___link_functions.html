function AntennaGain ( in d,
in frequency )

Antenna types are circular aperture or horn, omnidirectional or dipole, and helical. Omnidirectional antenna have a fixed gain of 1.5, circular and helical are functions of the geometry and frequency.

Type AntennaGain to run a demo for a circular aperture.

Form

 d = AntennaGain;  % data structure
 g = AntennaGain( d, frequency )

Inputs

 d             (.)   Antenna data structure
                       .type             (1,:) Type string
                       .area             (1,1) Area (circular or horn), m^2
                       .effIllumination  (1,1) Illumination efficiency
                       .effSpillover     (1,1) Spillover efficiency
                       .effSurfaceFinish (1,1) Surface finish efficiency
                       .effOhmic         (1,1) Ohmic losses
                       .effImpedance     (1,1) Impedancy mismatch losses
                       .eff              (1,1) Miscellaneous
                       .n                (1,1) Number of turns (helical)
                       .c                (1,1) Circumference (helical)
                       .s                (1,1) Spacing (helical)
 frequency    (1,:)  Frequency (GHz)

Outputs

 g            (:)    Gain (dB)

References: Maral, G. and M. Bousquet. (1998) Satellite Communications Systems. Wiley. pp. 14-17

References and(), AntennaGain(), DefaultDataStructure(), Frequency(), Illumination(), and isempty().

Referenced by AntennaGain(), DataRateGain(), LinkAnalysisTool(), and LinkBudget().

The quaternion is from planet fixed to body. Leave the one empty that you do NOT want to compute. In other words if b is empty it will compute q. The antenna boresight is defined as the local z-axis.

                 .lat (1,2) [min max] latitude (deg)

                 .n   (1,1) Number of users in zone

 nP      (1,1)   Number of orbit periods to analyze

Outputs
