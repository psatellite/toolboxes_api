Princeton Satellite System's MATLAB toolbox products include the Spacecraft Control Toolbox (SCT), Aircraft Control Toolbox (ACT), and Core Control Toolbox (CCT). The toolbox is organized into modules by functionality, for example Math and Airships. The documentation, function API and published demos for each module are linked below.
