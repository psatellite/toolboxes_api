DFIGSim.m File Reference

Simulate a doubly fed inductance generator. More...

Detailed Description

Published Demo

Doubly fed induction generators are popular for large wind turbines. Doubly fed means we feed both the rotor and stator. This demo is done with both rotor and stator in Direct Quadrature (DQ) coordinates. The controller computes an equilibrium solution then employs a linear quadratic regulator about that solution. This controller controls reactive and active power.
