EarthOrbitDisturbances.m File Reference
Sail Module » Sail Demos

Demonstrate the solar sail disturbance model in Earth orbit. More...

Detailed Description

Published Demo

The orbit is a circular equatorial orbit at 8000 km around the Earth. The sail is always sun-pointing. This demo is the same as the built-in demo in SailDisturbance.

Functions demonstrated: QSunPointing SailDisturbance

Since version 7.
