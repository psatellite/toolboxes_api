BallastMass2Axis.m File Reference

Attitude control of a sail using ballast masses in-plane. More...

Detailed Description

Published Demo

Demonstrates control of a single axis using a simple PID controller.
This uses a fixed-rate dynamics model that is suitable for masses that 
are controlled by stepper motors. 


Demonstrates:
PlateWithMasses
PIDMIMO
FMovingBody and FCoreAndMoving
TorqueToCM
CMToMassPositions
See also
AC, PIDMIMO, QTForm, QZero, Plot2D, Cross, RK4,
CMToMassPositions, TorqueToCM, FCoreAndMoving
