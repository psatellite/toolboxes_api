IsothermalUKF.m File Reference

A UKF that estimates thermal parameters for an isothermal model. More...

Detailed Description

Published Demo

The measurement is the average of all spacecraft temperatures. The user can select which parameters to estimate. The demonstration is for geosynchronous orbit. An internal heater has a diurnal (orbit frequency) heating pattern.
