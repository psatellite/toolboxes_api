Introduction

This tool allows you to load simulation results and create any number of plots of the data. You can save your plot preferences in a template. The parts of the Plotting Tool are explained below.

The Plotting Tool GUI

  • Control buttons: The buttons along the bottom control the window. You should begin by loading a sim data file, there should be examples in the folder even if you have not run any simulations yet. Then you can load a template, which contains various variable and plotting preferences. There should also be sample templates included with the PlottingTool. The other buttons allow you to manage the variable list and save your own templates.

You can load multiple sim data files. They will all use the same template.

  • Variables: The left side of the PlottingTool is devoted to the variables in the sim results file. Here you can see a list of the variables, and view information and plots of individual variables. Just click on a variable to view it.

The 'Group Variables' and 'Display Filter' allow you to modify the list of variables that you see according to your own preferences. When you load a template the variables will be filtered as defined in the template.

  • Animation pane: A pane in the middle of the Plotting Tool allows you to view animations of 3D variables. You must have previously defined some 3D variables, which will then appear in the popup menu in this pane. You can set the start and stop times for the animation, as well as the speed.
  • Plot definition: The right hand side of the Plotting Tool is devoted to user-defined plots. Use the buttons at the top to create or delete plots. These changes will be reflected in the popup menu. Use the 'Plot Time History' button at the bottom to generate the plot currently selected in the popup menu.

The check boxes allow you to select variables and satellites for your plot. Any combination of variables and satellite can be plotted using the subplot and line type information you select in the small popup menus.

Derivation functions

You may derive additional data from the variables included in the sim data file. The 'Derive Data' button brings up an additional GUI which can be used to specify a new type of derived data. Several derivation functions are included with the PlottingTool and additional functions may be defined by the user. You can select any of the previously defined variables as inputs for a derivation function.

Templates

All the changes you make to a sim data file, including defining and deriving new variables and figures, can be saved in a template. This template can be applied to any other desired sim data files as well. In this sense, templates really capture user preferences. Use the 'Load Template' and 'Save Template' buttons to save or create new templates and to apply a template to a sim data file.
