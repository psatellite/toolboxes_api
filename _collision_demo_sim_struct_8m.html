◆ CollisionDemoSimStruct()

blank line function CollisionDemoSimStruct ( in duration)

Specifies two spacecraft and four orbits and initial drift is removed. The reconfiguration is first commanded with a small size which causes it to be rejected, and then again with a larger size for which it is approved. To just check the CollisionMonitor output set the duration to 600 seconds.

Form

sim = CollisionDemoSimStruct(duration)

Inputs

duration (1) Duration, seconds (optional)

Outputs

sim (.) Simulation data structure to be input to DFFSimulation
