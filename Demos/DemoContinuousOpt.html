A non-impulsive optimization demo

An optimization using OptimizePolicyContinuous, which trades fuel for mission duration based on the input weighting factor.

This demo is off by default due to its long run time; to run the demo, first open it and change the value of the flag.

See also: IntegrateBurnPolicyContinuous, AccelFromPolicyContinuous, PlotTrajectoryContinuous, PlotDiagnosticContinuous

%--------------------------------------------------------------------------
%   Copyright (c) 2019 Princeton Satellite Systems, Inc.
%   All rights reserved.
%--------------------------------------------------------------------------

% run the full optimization? (default: false)
runOpt = false;

dSim = IntegrateBurnPolicyContinuous; % Simulation data struct
dAccel = AccelFromPolicyContinuous; % Acceleration data struct

% Initial and final orbits
dSim.el0 = [7000 0 0 0 0.01 0];
dSim.el1 = [42164 0 0 0 0 0];
dAccel.el1 = dSim.el1;

% Simulation time
dSim.tMax = 1e8; % Seconds
dAccel.tMax = dSim.tMax;

% Spacecraft parameters
dAccel.aMag = 1e-6; % Accel. km/s/s
dAccel.Isp = []; % Isp, seconds

% Discretized burn power angles, initial guess
initialGuess = [pi pi;-0.4*pi 0.4*pi];

% Set up optimization balance
%fac = 0.05; % Takes 70 minutes. 4.467 km/s, 4.42e6 seconds (low thrust spiral).
%fac = 0.0005; % Takes 12 hours. 3.787 km/s, 6.37e7 seconds (quasi-Hohmann).
%fac = 0.005; % Takes 6 hours. 3.824 km/s, 3.71e7 seconds.
fac = 0.015; % Takes 6 hours. 3.935 km/s, 2.24e7 seconds.

odeOpts = odeset( 'AbsTol', 1e-8, 'RelTol', 1e-6 ); % Trade simulation speed for accuracy

optOpts	= optimset(...
  'maxiter',1200,...
  'MaxFunEvals',12000); % Optimization options

% Plot initial guess
disp('Compute and plot the initial guess...')
OptimizePolicyContinuous(initialGuess,2,fac,dSim,dAccel,odeOpts,optOpts);
