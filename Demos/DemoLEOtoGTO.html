LEO to GTO single-rev impulsive demo

This is a transfer with a single optimal burn at apogee.

See also: OptDIH, TargetElementsImpulsiveAnalytic, OptimizeImpulsiveLVLH

% Start: LEO
inc    = 28.6*pi/180;
elInit = [6378+500 inc 0 0 0 2*pi*rand(1)];

% Highly eccentric GTO orbit
sma = 24582;
ecc = 42164/sma - 1;
fracI = 0.2; % fraction of inclination to remove
elFinal = [24582 (1-fracI)*inc 0 0 ecc 0];
PlotTwoOrbits(elInit,elFinal)

% Analytic Solution
[DV, dv1, dv2, et, Di1] = OptDIH( 0, elInit(1), elFinal(5), elFinal(1), -fracI*inc );
fprintf('\n\nAnalytic DV: %g km/s\n',DV)

Implement with a DV limit per rev

burnMaxDeltaV = 0.15;
burnsL = BreakImpulsesByDeltaV(burns1,burnMaxDeltaV,elInit);
PlotImpulsiveTrajectory(elInit, burnsL, elFinal,'lvlh');
titleStr = sprintf('DV Limit = %.2g km/s',burnMaxDeltaV);
title(titleStr)

return;

Loop

M0 = linspace(0,2*pi,50);
dVs = zeros(1,length(M0));
d.doPlot = false;
TimeDisplay( 'initialize', 'Optimize Burns', length(M0) );
for k = 1:length(M0)
  TimeDisplay( 'update' );
  elInit(6) = M0(k);
  [burns2] = OptimizeImpulsiveLVLH(elInit,elFinal);
  %[burns2,elOut,opt] = OptimizeElementsImpulsiveSearch(elInitial,elFinal,burns0,d);
  dVs(k) = sum(burns2(:,1));
end
TimeDisplay( 'close' )
Plot2D( M0, dVs, 'Initial Mean Anomaly', 'Optimal DV', 'Variation with Start Angle' )
