A non-impulsive burn demo of a planar, Hohmann-like transfer

Assumes a constant, low acceleration. Shows the result (trajectory, delta-V) of a given input and output orbit and discretized burn policy. A burn policy is a time-dependent true anomaly arc over which apoapsis, eccentricity, and plane are corrected. Burn policy is discretized in time and interpolated linearly.

This is a demonstration of how IntegrateBurnPolicyContinuous and AccelFromPolicyContinuous can be used to simulate a planar Quasi-Hohmann, 0.5% different from analytic. Takes 2 min.

See also: IntegrateBurnPolicyContinuous, AccelFromPolicyContinuous, PlotTrajectoryContinuous, PlotDiagnosticContinuous

dSim = IntegrateBurnPolicyContinuous; % Simulation data struct
dAccel = AccelFromPolicyContinuous; % Acceleration data struct

% Do not report diagnostic quantities
boolDiagnostic = false;

% Initial and final orbits
dSim.el0 = [7000 0 0 0 0.01 0];
dSim.el1 = [42164 0 0 0 0 0];
dAccel.el1 = dSim.el1;

% Simulation time
dSim.tMax = 5e7; % Seconds
dAccel.tMax = dSim.tMax;

% Spacecraft parameters
dAccel.aMag = 1e-5; % Accel. km/s/s
dAccel.Isp = []; % Isp, seconds

% Discretized burn power angles, linear in time
dAccel.ap = [10*pi/180]; % apoapsis correction, at periapsis
dAccel.per = [0 0 0 0 [1 1]*5*pi/180]; % Periapsis correction, at apoapsis. Turns on after some time.
dAccel.inc1 = [0]; % Plane correction at node 1
dAccel.inc2 = [0]; % Plane correction at node 2

acc = @AccelFromPolicyContinuous;
[x,t,out] = IntegrateBurnPolicyContinuous(dSim,acc,dAccel,[],boolDiagnostic);
PlotTrajectoryContinuous(x,t,dSim.el1);
tDuration = max(t(x(7,:)<(1-1e-2)*x(7,end)));
[tNum,~,tUnit] = TimeLabl(tDuration);
fprintf('Time to expend 99 pcent of dV is %.3g %ss. \n',tNum,tUnit);
fprintf('Delta-V via the integrator is %.3g km/s. \n',x(7,end));

[dv, dv1, dv2] = DVHoh(dSim.el0(1),dSim.el1(1),VOrbit(dSim.el0(1)));
fprintf('Hohmann: %.3g %.3g %.3g. \n',dv,dv1,dv2);

if boolDiagnostic
PlotDiagnosticContinuous(x,t,out.asVec,dAccel);
end
