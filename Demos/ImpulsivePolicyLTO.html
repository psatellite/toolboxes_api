GTO to LTO, impulsive multi-rev policy demo

With no plane change, in this case the only policy needed is per.

See also PropagateBurnPolicyImpulsive, PlotElementsImpulsive

data = PropagateBurnPolicyImpulsive;

initialRadius = 7000;
GEORadius = 42164;
moonOrbRad = 385e3;
GTOSMA = (initialRadius + GEORadius)/2;
GTOEccentricity = (GEORadius - initialRadius) / 2 / GTOSMA;
LTOSMA = (initialRadius + moonOrbRad)/2;
LTOEccentricity = (moonOrbRad - initialRadius) / 2 / LTOSMA;
data.el0 = [GTOSMA 0 0 0 GTOEccentricity 0];
data.el1 = [LTOSMA 0 0 0 LTOEccentricity 0];

% burn at perigee only
data.ap = 0;
data.per = 0.005;
data.inc1 = 0;
data.inc2 = 0;
data.tolAp = 0.01;
data.tolEcc = 1; % uncontrolled

[burns, out] = PropagateBurnPolicyImpulsive(data);
PlotElementsImpulsive(data.el0,burns,[],data.el1);
subplot(4,2,2)
title(sprintf('delta-V: %g km/s\n',SumBurnsDV(burns,1)))
