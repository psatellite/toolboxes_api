Show ISS and NanoSat models with AnimateSpacecraft

Loads two mat-files, Nanosat.mat and ISS.mat. Updates the display in a loop for one orbit to create an animation.

------------------------------------------------------------------------
See also: ISSOrbit, RVOrbGen, El2RV, RV2El, QLVLH, AnimateSpacecraft
------------------------------------------------------------------------
%--------------------------------------------------------------------------

g1        = load('Nanosat');
g2        = load('ISS');
el1       = ISSOrbit;
el2       = el1 + [0 0 0 0 0 0.000001];
[r1,v1,t]	= RVOrbGen(el1);
[r2,v2]   = El2RV(el2);
v2(2)     = v2(2) + 0.000001;
el2       = RV2El(r2,v2);
[r2,v2]   = RVOrbGen(el2,t);
q1        = QLVLH(r1,v1);
q2        = QLVLH(r2,v2);

   pE          = load('EarthHR');

AnimateSpacecraft( 'initialize', g1, g2, r1(:,1), v1(:,1), q1(:,1), r2(:,1), v2(:,1), q2(:,1) );
    AnimateSpacecraft( 'add planet', pE.planet );
    AnimateSpacecraft( 'track with camera',2,50,20);
m = AnimateSpacecraft( 'update', t, r1, v1, q1, r2, v2, q2 );
