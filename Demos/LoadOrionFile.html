Demonstrate converting an OBJ model to CAD.

This converts an OBJ file for Orion into a CAD data structure. The OBJ file must only contain triangles. The resulting file can be used to build other CAD files by loading it as a subassembly.

The file loaded is Orion_spacecraft_game_R.obj, the assembly is 'OrionNew':

BuildCADModel( 'add subassembly', 'OrionNew' );

See also: LoadCAD, SaveStructure
