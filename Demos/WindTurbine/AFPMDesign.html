Design a dual rotor axial flux generator.

The generator is coreless, i.e. has no backing steel for the stator windings. This is a lightweight, compact generator design.

See also DualRotorDesign, WindTurbinePower

Wind turbine design

%--------------------
v = 13; % Design speed for wind (m/s)

Compute wind turbine power

We want 11 kW output

h = 30;      % m
temp = 10;   % deg-C
lambda = 10; % Tip speed ratio
[power, torque, omega] = WindTurbinePower( 0.25*pi*5^2, [], v, h, temp, lambda );

Setup data structure for dual rotor design

d               = DualRotorDesign;
d.rO            = 0.27; % outer radius, m
d.n             = 24;   % poles
d.jPeak         = linspace(4e6,6e6); % A/m^2
d.rho           = 1.7e-8;   % mho/m
d.bPeak         = 0.65;     % T
d.wireD         = 0.000355; % m
d.omega         = omega;    % rad/s
d.t             = 0.01;     % m
d.hM            = 0.01;     % m
d.densityMagnet = 7400.0; % http://www.magnetsource.com/Solutions_Pages/rareerth.html
d.densityCopper = 8960.0; % kg/m^3

Perform the dual rotor design

DualRotorDesign( d );
p = DualRotorDesign( d );

Results

h  = find( p.pOut >= power );
j  = h(1);

fprintf('Power            = %8.2f kW\n',p.pOut(j)/1000)
fprintf('Eff              = %8.3f \n',p.eff(j))
fprintf('Design Spin Rate = %8.2f (rad/s)\n',omega)
fprintf('Mass Cu          = %8.2f kg\n',p.mCopper)
fprintf('Mass Neodymium   = %8.2f kg\n',p.mMagnet)
fprintf('Mass Active      = %8.2f \n',p.mActive)
