An inter-LEO non-impulsive burn demo

Demonstrates a continuous transfer between to LEO orbits using IntegrateBurnPolicyContinuous. For a GEO transfer, see DemoContinuousThrust.

Output is then compared to a burn policy with keepPer = true, which results in a slightly longer, slightly less DV solution.

% See also: IntegrateBurnPolicyContinuous, AccelFromPolicyContinuous,
% PlotTrajectoryContinuous, PlotDiagnosticContinuous

%--------------------------------------------------------------------------
%   Copyright (c) 2019 Princeton Satellite Systems, Inc.
%   All rights reserved.
%--------------------------------------------------------------------------

dSim = IntegrateBurnPolicyContinuous; % Simulation data struct
dAccel = AccelFromPolicyContinuous; % Acceleration data struct

% Do not report diagnostic quantities
boolDiagnostic = false;

% Initial and final orbits; nodes are aligned
dSim.el0 = [7000 28*pi/180 0 0 0.001 0];
dSim.el1 = [7500 30*pi/180 0 0 0 0];
dAccel.el1 = dSim.el1;

% Simulation time
dSim.tMax = 3.5e6; % Seconds
dAccel.tMax = dSim.tMax;

% Spacecraft parameters. If Isp is empty then accel is constant.
dAccel.aMag = 1e-6; % Accel. km/s/s
dAccel.Isp = []; % Isp, seconds

% Discretized burn power angles, linear in time
dAccel.ap = [30*pi/180]; % apoapsis correction, at periapsis.
dAccel.per = [30*pi/180]; % Periapsis correction, at apoapsis.
dAccel.inc1 = [30*pi/180]; % Plane correction at node 1.
dAccel.inc2 = [30*pi/180]; % Plane correction at node 2.

% tighten tolerances
dAccel.tolAp = 0.001;
dAccel.tolInc = 0.001;
dAccel.tolEcc = 0.001;

a = @AccelFromPolicyContinuous;
[x,t,out] = IntegrateBurnPolicyContinuous(dSim,a,dAccel,[],boolDiagnostic);
PlotTrajectoryContinuous(x,t,dSim.el1);
PlotElementsContinuous(x,t,dSim.el1);
[tNum,~,tUnit] = TimeLabl(t(end));
fprintf('Time to reach elements is %.3g %ss. \n',tNum,tUnit);
fprintf('Delta-V via the integrator is %.3g km/s. \n',max(x(7,:)));

if boolDiagnostic
PlotDiagnosticContinuous(x,t,out.asVec,dAccel);
end

Compare to perigee-preserving pointing of apse burns

This maintains a higher eccentricity for longer during the transfer. This takes a little bit less DV and a little more time.

dAccel.keepPer = true;
[x2,t2,out2] = IntegrateBurnPolicyContinuous(dSim,a,dAccel);
[tNum2,~,tUnit2] = TimeLabl(t2(end));
fprintf('Time to reach elements is %.3g %ss. \n',tNum2,tUnit2);
fprintf('Delta-V via the integrator is %.3g km/s. \n',max(x2(7,:)));

% Compare the elements directly
elA = RVSet2El(x(1:3,:),x(4:6,:));
elB = RVSet2El(x2(1:3,:),x2(4:6,:));
Plot2D(t/86400,elA(:,[1 2 5])','Time (days)',{'SMA','inc','ecc'},'Policy Comparison')
subplot(3,1,1);
hold on; plot(t2/86400,elB(:,1)');
subplot(3,1,2);
hold on; plot(t2/86400,elB(:,2)');
subplot(3,1,3);
hold on; plot(t2/86400,elB(:,5)');
