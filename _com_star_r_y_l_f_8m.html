Design and test the low frequency roll/yaw control system. More...

Detailed Description

Published Demo

The script has three parts:

  1. Investigate the orbit rate disturbance attenuation assuming that the orbit rate disturbance is caused by an inertially fixed torque.
  2. Select gains and look at the torque transmission plots. The first set are closed loop. The second set are open loop (for reference purposes.)
  3. Simulate the response to the orbit rate disturbance.
  4. Must run ComStarDist.m first (saves/updates DistModel.mat).

Saves the control matrices in RYC.mat.
