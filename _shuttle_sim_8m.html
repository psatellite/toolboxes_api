ShuttleSim.m File Reference

Simulate the space shuttle model. More...

Detailed Description

Published Demo

We rotate the shuttle bay doors and the attached robot arm using the body hinge structure of the CAD model. The shuttle model is loaded from a mat-file. This script shows how to interact with DrawSCPlugIn in a simulation loop.
